# isaac_demo

A combined set of demo working with Isaac SIM on a workstation and Isaac ROS on a NVIDIA Jetson AGX Orin

## Hardware required

Workstation:

1. x86/64 machine
2. NVIDIA Graphic card with RTX
3. Display
4. Keyboard and Mouse

NVIDIA Jetson:

1. NVIDIA Jetson AGX Orin
2. Jetpack 5.0.2

Tools:

1. Router
2. eth cables
